How to run the Heliosphere simulation given the problem generator and input file on Discovery:

1). Clone the repository to your local directory

2). Navigate to the Athena++Dedner Directory

3). Load these modules: 
module load mpi/mpich-x86_64
module load openmpi/5.0

4). Configure the makefile:
python configure.py --prob=MHDHelioSphericalBCAMR2D -b -hdf5

5). Type "make all" to produce the configured makefile tailored to our problem

6). after the makefile finishes, move to the 'bin' directory with the new athena executable

7). Run the makefile with one of the input files, for me this looks like:
./athena -i /dartfs-hpc/rc/home/r/f007h2r/Phys118FinalProject/Athena++Dedner/inputs/mhd/athinput.MHDHelioSphericalBCSMRcp10ch7


(note the input files can be found in the 'inputs' folder in the Athena++Dedner directory
