How to run the 2D Heliosphere simulation given the problem generator and input file:

1). Navigate to the Athena++Dedner directory

2). Configure the makefile using terminal/python: python3 configure.py --prob=MHDHelioSphericalBCAMR -b --flux=lhlld -hdf5

3). Once Athena++ alerts you it has been successfully configured, run "make all" in your terminal. The source files should begin compiling. After about a minute, you should see a successful compilation!

Note: if you do not have HDF5 installed, you will need it to get the proper output format. If you do have it installed but Athena++ is unable to recognize its location, look through the 'HDF5 Makefile Configuration' text file!

4). Now we need to run the generated executable with our input file. A new /bin directory will be created after successful compilation. Navigate to this directory, where you will then see the athena.exe executable. All we need to do is supply the input file to use. Run this in your terminal:

./athena -i YOURCUSTOMPATH/Athena++Dedner/inputs/mhd/athinput.MHDHelioSphericalBCAMR2D

Replace YOURCUSTOMPATH with your device's custom path to the Athena++Dedner directory. 

The code should then run and produce output files :). You can customize the input file to edit the strength of cleaning, etc. The output format are HDF5 files. These are great to view in the Paraview software, however I will no go into depth on how to use the program here!
